\section{3D视窗}

\begin{intro}
    3D视图用于与3D场景交互以用于各种目的，例如建模、动画、纹理绘制等。
\end{intro}

\subsection{选择-菜单>选择}
\begin{itemize}
    \item 选择物体：「左键」物体
    \item 取消选择：「左键」空白处
    \item 全选：「A」
    \item 取消全选：「Alt+A」
\end{itemize}

\subsection{视窗导航-菜单>视图}
\begin{itemize}
    \item 旋转：「中键」或「Gizmo球体」或「数字键4862」
    \item 平移：「shift+中键」或「Gizmo手掌」
    \item 缩放：「滑轮」或「Gizmo放大镜」或「数字键-+」
    \item 查看选中对象：「数字键.」
    \item 摄像机视图：「数字键0」或「Gizmo摄像机」
\end{itemize}
